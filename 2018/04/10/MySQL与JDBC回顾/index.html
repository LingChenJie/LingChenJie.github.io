什么是JDBC

JDBC（Java DataBase Connectivity）就是 Java 数据库连接，JDBC 是用 Java 语言向数据库发送 SQL 语句.JDBC是接口，而JDBC驱动才是接口的实现，没有驱动无法完成数据库连接！每个数据库厂商都有自己的驱动，用来连接自己公司的数据库
